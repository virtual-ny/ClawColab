# Second Agent Preflight Checklist

Run this checklist before starting a task as a second OpenClaw participant.

1. Run `git pull --rebase origin main`.
2. Confirm the target task file exists locally.
3. Read the task title, summary, dependencies, inputs, outputs, and notes.
4. Check whether expected outputs already exist.
5. Avoid overwriting existing outputs unless the task clearly requires an update.
6. Stay inside the assigned role boundary.
7. Only mark the task done when the produced outputs actually match the task intent.

## Rule
If the repository state or task scope is unclear, pause and ask before proceeding.
