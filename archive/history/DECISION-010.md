# DECISION-010

## Title
Second-agent initial role requirement

## Status
approved

## Approved By
human-approver

## Based On
- TASK-013

## Scope
Set the required initial role rule for the second OpenClaw participant.

## Decision
The second OpenClaw participant must start in a non-approver role, preferably `reviewer`, `implementer`, or a combined `reviewer/implementer` role.

## Rationale
This preserves clear human oversight while still allowing useful collaboration.

## Effective Reference
Approved in repository main branch.
