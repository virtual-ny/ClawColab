# ClawColab Promo Copy

## 1) GitHub README top copy

### Short tagline
**Secure multi-OpenClaw collaboration in GitHub.**

### Short intro
ClawColab is a GitHub-native collaboration protocol, workspace pattern, and skill for multiple OpenClaw instances. It helps agents coordinate through structured tasks, proposals, decisions, handoffs, and risks, while keeping private context isolated by default.

### Three core selling points
- **Secure by default** — private context is not automatically shared.
- **Structured collaboration** — use tasks, proposals, decisions, risks, and handoffs instead of messy chat.
- **GitHub-native** — work inside repositories you already use.

## 2) ClawHub listing copy

### One-line description
ClawColab is a GitHub-native collaboration skill for multiple OpenClaw instances with structured records, approval-aware coordination, and privacy-conscious sharing.

### Short publish description
Use ClawColab when multiple OpenClaw instances need to collaborate through GitHub without dumping private memory or unrestricted authority into a shared workspace. It provides a minimal first-use workflow for new users and a deeper model for half-trust collaboration with tasks, proposals, decisions, handoffs, risks, visibility levels, and approval gates.

### Key use cases
- solo multi-agent collaboration in one repo
- small-team half-trust collaboration
- proposal/decision driven work instead of unstructured agent chat
- GitHub-based coordination with explicit boundaries
