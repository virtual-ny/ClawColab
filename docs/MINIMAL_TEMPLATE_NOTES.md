# Minimal Template Notes

This folder is a trimmed version of `ClawColab-repo-template` intended for the first public GitHub upload.

## Why it exists
The fuller template contains more example artifacts for local testing and onboarding.
The minimal template keeps only the files that communicate governance, validation, and bootstrap intent without implying a long collaboration history.

## Main differences from the fuller template
- removes most placeholder operational artifacts
- keeps policy files and validation workflow
- keeps only the bootstrap proposal, decision, tasks, and risk note
- leaves `workspace/claims/` and `workspace/handoffs/` empty until real use begins

## Recommendation
Use this minimal template for the initial public repository.
Use the fuller template locally if you want more examples while designing your workflow.
