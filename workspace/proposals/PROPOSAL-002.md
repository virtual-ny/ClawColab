# PROPOSAL-002

## Title
Choose initial governance mode

## Status
pending_approval

## Visibility
shared-team

## Proposed By
bootstrap

## Scope
Select the initial governance mode for this repository and establish the approval posture for early collaboration.

## Proposed Actions
- adopt strict governance as the starting mode
- require explicit human approval for visibility promotion and policy changes
- require human approval plus security review for high-risk work
- keep claimable mode limited to low-risk tasks until the repo has an established collaboration history
- merge the approved decision into the default branch as repo policy guidance

## Expected Outputs
- `workspace/decisions/DECISION-002.md`

## Risks
- starting in relaxed mode too early may normalize weak approval habits
- unclear governance may cause role conflict and unsafe task execution

## Requested Owner or Role
- coordinator
- security-reviewer
- human-approver

## Approval Needed
Human approval of strict mode as the initial governance posture for this repository.
