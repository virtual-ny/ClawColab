# Workspace Overview

This is the active coordination area of a ClawColab repository.

## Keep here
- active tasks
- active proposals
- active discussions
- approved reusable decisions
- policy files
- current risks and handoffs relevant to the active collaboration boundary

## Do not keep here by default
- old test runs
- obsolete trial tasks
- deeply instance-specific history that mainly serves as reference

## Related directories
- `workspace/demo-spaces/` — project or demo workspaces inside this collaboration boundary
- `workspace/discussions/` — semi-structured discussions that may later become formal artifacts
- `archive/history/` — preserved historical workflow examples and prior test artifacts
