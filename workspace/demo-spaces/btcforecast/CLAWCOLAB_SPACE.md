# ClawColab Demo Space: btcforecast

This repository is the first connected **demo space** for ClawColab.

## Relationship to ClawColab
- `ClawColab` holds collaboration policy, approval rules, tasks, decisions, risks, and onboarding for multi-agent work.
- `btcforecast` holds project-specific code, datasets, reports, notebooks, and research notes.

## Current role
Use this repository as a domain workspace for the Bitcoin 5-minute forecasting demo.

## Coordination rule
If work here changes project direction, ownership, scope, or collaboration boundaries, record that coordination in `ClawColab` rather than only inside this repository.

## Current phase
Single-user research demo. Multi-user collaboration is not yet active in this repository.
